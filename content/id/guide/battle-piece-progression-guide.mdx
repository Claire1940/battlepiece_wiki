<Steps>
  <Step num="1" title="Luncurkan Game Resmi">
    Akses game melalui [Halaman Game Roblox Resmi](https://www.roblox.com/games/98662768290911/Battle-Piece) untuk menghindari tiruan duplikat atau tidak resmi. Bergabunglah dengan [Komunitas Studio Multiverse](https://www.roblox.com/communities/903561179/Studio-Multiverse) untuk tetap mendapatkan info terbaru tentang pembagian kode dan catatan patch.
  </Step>
  <Step num="2" title="Farming Bandit untuk Level Inti">
    Mulailah perjalanan Anda dengan menargetkan bandit level rendah. Fokus pada mempelajari jangkauan serangan, ritme cooldown, dan gerakan dasar. Farming bandit adalah cara teraman untuk membangun pengalaman dasar Anda.
  </Step>
  <Step num="3" title="Tingkatkan Kekuatan dan Senjata">
    Seiring naik level, investasikan pada kekuatan dan senjata yang menawarkan kerusakan area (AoE). Ini memungkinkan Anda untuk membasmi banyak musuh secara bersamaan, meningkatkan pengalaman per jam Anda secara signifikan.
  </Step>
  <Step num="4" title="Latih Penempatan Jarak di Server Privat">
    Setelah Anda mencapai level mid-game, beralihlah ke server privat untuk melatih teknik penempatan jarak dan penghindaran tingkat lanjut tanpa gangguan dari pemain musuh.
  </Step>
  <Step num="5" title="Dorong ke Batas Level">
    Dedikasikan level terakhir Anda untuk tempat farming hasil tinggi dan upaya raid awal. Mencapai Level 600 akan membuka potensi penuh dari build Anda untuk PvP kompetitif.
  </Step>
</Steps>
