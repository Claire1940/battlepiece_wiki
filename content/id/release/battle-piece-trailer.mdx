<Section icon={Target} title="Strategi Raid & PvP">

Progres akhir game di Battle Piece berkisar pada penguasaan raid dan menguji keterampilan Anda dalam pertandingan PvP. Kedua aktivitas tersebut memerlukan pemahaman mendalam tentang mekanik pertempuran, tetapi menuntut pendekatan taktis yang sama sekali berbeda.

| Langkah Persiapan | Persyaratan Raid | Persyaratan PvP |
| :--- | :--- | :--- |
| **Target Level** | Mendekati batas level **600** | Direkomendasikan Level **400+** |
| **Prioritas Loadout** | Damage area luas dan kelangsungan hidup | Mobilitas tinggi dan potensi burst |
| **Pengaturan Jaringan** | Server privat untuk menghindari lag raid | Server ping rendah untuk akurasi input |
| **Koordinasi Tim** | Prioritas tinggi untuk pembersihan grup | Opsional, fokus pada penempatan posisi individu |

<Callout type="tip" title="Penempatan Posisi Pertempuran">
Baik di PvP maupun raid, menyerang dengan serakah akan menyebabkan kekalahan cepat. Pelajari cara memancing cooldown lawan sebelum melakukan kombo burst berdamage tinggi Anda. Gunakan keterampilan mobilitas Anda secara defensif untuk mengatur ulang pertempuran saat diperlukan.
</Callout>

Untuk raid, koordinasi dengan tim Anda sangatlah vital. Pastikan grup Anda memiliki keseimbangan crowd control untuk menjaga musuh tetap berkumpul dan dealer damage tinggi untuk menghabiskan bilah kesehatan bos. Di PvP, fokuslah sepenuhnya pada penempatan posisi individu, mempelajari jangkauan tepat dari senjata lawan Anda, dan menghukum serangan mereka yang meleset.

</Section>
