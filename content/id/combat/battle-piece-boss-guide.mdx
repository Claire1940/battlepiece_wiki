<Section icon={Target} title="Strategi Boss Raid Langkah-demi-Langkah">

Menaklukkan boss raid di Battle Piece membutuhkan koordinasi, waktu, dan pengaturan server yang tepat. Latensi tinggi dapat merusak input pertempuran, membuat penghindaran berbasis waktu hampir mustahil. Ikuti strategi langkah-demi-langkah ini untuk memaksimalkan peluang keberhasilan Anda selama pertempuran raid.

<Steps>
  <Step num="1" title="Optimalkan Level dan Statistik Anda">
    Jangan langsung mencoba raid setelah membukanya. Grind bandit dan boss kecil sampai Anda mendekati batas level 600 untuk memastikan kolam darah Anda dapat bertahan dari serangan yang tidak disengaja.
  </Step>
  <Step num="2" title="Amankan Server Privat">
    Ping tinggi adalah penyebab utama kegagalan raid. Gunakan server privat untuk menstabilkan koneksi Anda, memastikan input hindaran dan serangan Anda terdaftar tanpa penundaan.
  </Step>
  <Step num="3" title="Bersihkan Gelombang Kroco Awal">
    Boss raid dilindungi oleh gelombang kroco yang kuat. Gunakan keterampilan PvE area luas untuk membersihkan musuh-musuh ini dengan cepat sebelum mereka mengepung dan membanjiri tim Anda.
  </Step>
  <Step num="4" title="Umpan dan Hukum Serangan Boss">
    Jaga jarak Anda dan tunggu boss melakukan serangan berat. Setelah animasi selesai, dekati dengan alat mobilitas, lepaskan kombo burst Anda, dan segera mundur.
  </Step>
</Steps>

Memilih jenis server yang tepat membuat perbedaan signifikan dalam eksekusi pertempuran. Tabel perbandingan di bawah ini menyoroti metrik kinerja antara opsi server.

| Jenis Server | Rata-rata Ping | Kepadatan Pemain | Kecepatan Farming Boss | Rekomendasi |
|--- |--- |--- |--- |--- |
| Server Publik | Tinggi | Tinggi | Lambat | Bagus untuk leveling santai |
| Server Privat | Rendah | Tidak ada | Cepat | Penting untuk raid dan boss |

</Callout>
